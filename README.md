# B&J Biscuit Sales Dashboard (Excel)

## Project Overview
This project presents an interactive Excel dashboard designed to analyze sales performance, customer behavior, and profitability for a biscuit company.

The dashboard transforms raw data into meaningful insights using advanced Excel features such as Pivot Tables, charts, and interactive filters.

## Dashboard Features

### Revenue Analysis
- Revenue distribution by product price category  
- High vs Low price comparison  

### Customer Insights
- Top 5 customers by revenue contribution  
- Age-group based revenue segmentation  

### Geographic Analysis
- Revenue share across different cities  
- Location-based filtering  

### Payment Behavior
- Revenue distribution by payment methods:
  - Cash  
  - Credit Card  
  - Debit Card  
  - Mobile Payment  

### Demographic Analysis
- Revenue breakdown by gender  
- Age group contribution
